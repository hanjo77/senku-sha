Installation Manual

To install Senku-Sha on your web server, follow these steps:

  1. Find the SQL script file doc/senku-sha-init-db.sql and run it on your mySQL server to create the necessary database and tables.
  2. Put all files inside your preferred folder of your web server.
  3. Edit the file inc/config.php to fit your server settings:
    • If you do not change the mail parameters, PHP will not be able to send e-mails to newly registered users.
    • If you have not changed anything to the SQL script, you will only have to change the server URL of the db settings.
  4. Optional: Edit the file js/config.js if you want to change any game settings.
